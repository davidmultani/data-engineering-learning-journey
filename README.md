# Data Engineering & Machine Learning Systems

This repository contains experiments, implementations, and project work related to **data engineering, machine learning, and production ML systems**.

The focus is on building practical systems for **data processing, modeling, and deployment**, while exploring modern tools used in real-world data platforms.

# Areas of Work

This repository covers several areas across the data ecosystem:

### Data Processing

* Python for data processing
* NumPy and Pandas
* Data cleaning and transformation
* Feature engineering
* Exploratory data analysis

### Statistics & Experimentation

* Statistical analysis
* Hypothesis testing
* A/B testing
* Data-driven decision frameworks

### Machine Learning

* Regression and classification models
* Decision Trees
* Random Forest
* Gradient Boosting
* Model evaluation and validation
* Hyperparameter optimization

### Natural Language Processing

* Text preprocessing pipelines
* TF-IDF and embedding-based models
* Transformer-based models
* Prompt engineering
* Retrieval Augmented Generation (RAG)

### Data Engineering

* SQL analytics and query optimization
* ETL pipelines
* Data ingestion from APIs
* Data quality validation
* Workflow automation

### Production ML Systems

* FastAPI model serving
* Docker containerization
* MLflow experiment tracking
* Model versioning
* Cloud deployment (AWS / GCP)

### Data Visualization

* Interactive dashboards
* Plotly / Streamlit
* Business analytics reporting
